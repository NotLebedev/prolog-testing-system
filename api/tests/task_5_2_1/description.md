## Упражнение 5.2.1
Создайте логическую программу, описывающую следующие свойства термов:

`my_list(X)` — "X является списком".

Выяснить, каково множество правильных ответов на следующие запросы, обращенные к построенным программам:

1. `? my_list(a.b.c.nil)`
2. `? my_list(a.X.nil)`
3. `? my_list(a.b)`
